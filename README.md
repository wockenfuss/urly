# Urly - URL Shortener

First Dev Bootcamp Rails project, ultra-bare bones

## Features (or lack thereof)

* Basic user accounts
* Raw HTML, no styling
* Allows users to add vanity url on creation. If no vanity url is present, the shortener generates a custom short url.
